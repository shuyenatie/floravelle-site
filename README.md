# 外贸网站搭建实践项目

## 项目简介
本项目是一个使用 AI 辅助搭建的外贸网站练习项目，主要用于学习外贸网站页面结构、产品展示、基础文案、搜索意图理解和 SEO/GEO 内容优化逻辑。

## 当前页面
- 首页 index.html
- 产品页 product.html
- 产品集合页 collections.html
- 工厂介绍页 factory.html
- OEM定制页 custom-oem.html
- 联系我们 contact.html

## 使用工具
- Codex
- GitHub

## AI参与方式
- 辅助拆解网站页面结构
- 辅助生成基础页面代码
- 辅助优化产品文案和页面表达
- 辅助整理用户可能关注的问题
- 辅助排查页面问题并迭代修改

## GEO/内容优化练习
- 练习产品卖点表达
- 练习用户搜索意图理解
- 练习页面标题、产品描述和FAQ内容整理
- 练习让网站内容更清晰、更容易被AI理解和引用

## 当前状态
个人学习项目，持续完善中。
